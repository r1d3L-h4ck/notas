### 1) Formatos de texto 

#### negrita  

```
    **texto negrita 1** 

    __texto negrita 2__ 
```
**texto negrita 1** 

**texto negrita 2** 

#### cursiva  

```
    *texto cursiva 1*   

    _texto cursiva 2_   
```
*texto cursiva 1*

 _texto cursiva 2_

#### negrita y cursiva 

```
    ***texto negrita y cursiva 1***     

    __texto negrita y cursiva 2___
```
***texto negrita y cursiva 1***

___texto negrita y cursiva 2___

#### tachado

```
	~~tachado~~
```
~~tachado~~

### 4) Listas

#### Desordenadas

```
    - elemento 1
    - elemento 2
    
    * elemento 1
    * elemento 2
    
    + elemento 1
    + elemento 2
```
- elemento 1
- elemento 2

* elemento 1
* elemento 2

+ elemento 1
+ elemento 2

#### Ordenadas

```
    1. elemento 1
        1. subelemento
        2. subelemento
    2. elemento 2
        1. subelemento
        2. subelemento
```
1. elemento 1
    1. subelemento
    2. subelemento
2. elemento 2
    1. subelemento
    2. subelemento
